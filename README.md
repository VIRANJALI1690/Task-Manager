# Terminal Based Task Manager using MongoDB

This is a Terminal-Based Task Manager built using MongoDB Shell (mongosh). It performs CRUD operations and filter queries to manage tasks efficiently.

## Features

- Create tasks
- View tasks
- Update tasks
- Delete tasks
- Filter tasks based on status and priority

## Technologies Used

- MongoDB
- MongoDB Shell (mongosh)
- GitHub

## How to Run

1. Open terminal
2. Run mongosh
3. Copy commands from commands.txt
4. Execute commands
